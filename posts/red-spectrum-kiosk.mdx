***Tech Stack:*** Raspberry Pi, Bash

## Advertisment Kiosk where raspberry pi points to webpage that is just an advertisement

- Installs chromium to raspberry pi and points to the advertisement webpage. 
- If turned off and on the kiosk will be pointing to the same website and display the needed page. 
- It can be updated from the CMS because it is set to refresh every minute.
